### `iOS 聊天简易Demo`

>
iOS TCP 连接通信，完成简单文本传输。支持多个客户端连接
>
供初学者学习

===

#### `实现`

- `Server BSDSocket`
- `Client AsyncSocket`
- `Xcode 8.0`
- `简单JSON格式`
